# sequelize-paper-trail-example

> A demo implementation of Sequelize Paper Trail module.

## Installation

```bash
git clone https://github.com/nielsgl/sequelize-paper-trail-example.git
cd sequelize-paper-trail-example
npm install
```

## Starting the app

```bash
gulp
```
